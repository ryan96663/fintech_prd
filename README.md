## 项目简介
### 解决什么需求
- 看不懂保险，不敢买，怕买错
	 保险条款专业壁垒高，普通用户面对重疾险、医疗险、意外险、寿险的分类无从下手。
- 多分保险，不了解保障覆盖情况
	 一个家庭可能持有多份保单，分散在不同保险公司，其保障重叠、存在未覆盖风险敞口较为常见。用户需要从综合角度给出量化分析和保障优化建议的综合管理平台，解决用户跨平台管理保单、并选购最合适的保单精准覆盖风险缺口。
- 理赔环节维权模糊、材料复杂、进度了解不清楚、流程对接复杂等问题
	 将理赔环节变成客户留存的手段，提供全线的理赔服务，为客户在每个环节提供精准指引和对接，实时检测了解理赔真实进度，减少用户报销门槛，提高理赔效率，有效增强用户对保险的信任度，提高用户复购率，构建保险的完整闭环服务。
### 核心功能

| 模块          | 做什么                                                                          |
| :---------- | :--------------------------------------------------------------------------- |
| **保障评估中心**  | 场景化问卷 → 6 维风险敞口计算 →可视化呈现（如雷达图）                                               |
| **智能方案生成**  | 评估报告 → 多产品组合匹配 → 预算约束优化 → 2-3 套方案横向对比                                        |
| **家庭管理**    | 成员档案 + 已有保障盘点 + 家庭 6 维雷达图（寿险/重疾/医疗/意外/养老/财产）                                 |
| **产品市场**    | 搜索/筛选/多产品对比→ 差异高亮 + 雷达图可视化表述                                                 |
| **交易管理**    | 订单确认 → 核保 → 支付 → 保单生成                                                        |
| **智能理赔**    | 一键发起 → AI 判断理赔类型 → 材料清单+上传 → AI 材料核验 → 6 节点进度追踪 → 赔付到账                       |
| **AI 智能客服** | 全局悬浮按钮，从基础知识到专业保险知识，从单一保险信息询问到家庭保障方案细则询问均，可提供用户具体业务信息讲解、专业方案分析与常识知识介绍等全方位解答。 |
